Shin-chan Character Counter
A fun, interactive web application that tracks character limits in real-time, featuring a Crayon Shin-chan theme.

Project Overview
This project is a simple yet functional character counter. It allows users to type messages while providing live feedback on how many characters they have used and how many are remaining before reaching the 50-character limit.

Features
Live Tracking: Character counts update instantly as you type.

Shin-chan Theme: Custom styling with a full-screen background image and a matching UI card.

Input Validation: Prevents users from typing more than the allowed 50 characters using the maxlength attribute.

Clean UI: Styled with CSS Flexbox for perfect centering and a modern look.

Tech Stack
HTML5: For page structure.

CSS3: For the layout, background effects, and color-coded counters.

JavaScript: For the real-time calculation logic.
